OA Schema Documentation

This document provides an overview of the OA Schema, including its entities, relationships, and namespaces.

Entities

The following entities are defined within the OA Schema:

  • Object: An abstract entity.
    • relatedTo → Morphism
  • Morphism: A directional relationship.
    • mapsTo → Object

Namespace

The following namespace is used in this schema:

urn:oa:schema:graph
